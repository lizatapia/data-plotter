# Data-plotter
Plot data from tiff and shape files

## Prerequisites
* [Python](https://www.python.org/downloads)
* [Pipenv](https://pipenv.pypa.io/en/stable/installation/#installing-pipenv)

## Setup
### Install dependencies
```shell
pipenv install
pipenv shell
```

## Run

Run plotter with mock data
```shell
python edumap.py
```
